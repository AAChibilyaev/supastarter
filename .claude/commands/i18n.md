Выполни команду i18n-инструмента для AACsearch. Запусти:

```bash
python3 packages/i18n/scripts/i18n.py $ARGUMENTS
```

Выведи результат напрямую.

## Справка по командам (если $ARGUMENTS пустой или "help"):

| Команда | Пример |
|---------|--------|
| `get <scope> <key>` | `get saas search.synonyms.title` |
| `set <scope> <key> <en> <de> <es> <fr> <ru>` | `set saas search.new.label "Label" "Label" "Etiqueta" "Étiquette" "Метка"` |
| `del <scope> <key>` | `del saas search.old.key` |
| `ls <scope> [prefix]` | `ls saas search.stopwords` |
| `check <scope> [prefix]` | `check saas search` |
| `mv <scope> <old> <new>` | `mv saas search.old.key search.new.key` |
| `cp <src_scope> <src_key> <dst_scope> <dst_key>` | `cp saas search.x marketing common.x` |

**Scopes:** `saas` · `marketing` · `shared` · `mail`

**Правило:** при любой работе с i18n-ключами в этом проекте — ВСЕГДА использовать этот инструмент.
Прямое редактирование JSON-файлов руками запрещено — только через скрипт.
