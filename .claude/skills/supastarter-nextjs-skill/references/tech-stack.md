# Tech Stack (Next.js)

Current supastarter Next.js stack (2026 sitemap aligned).

## Stack Summary

- **Framework**: Next.js App Router + React + strict TypeScript
- **Apps**: `apps/saas`, `apps/marketing`, `apps/docs`, `apps/mail-preview`
- **API**: Hono + oRPC (`packages/api`)
- **Database**: Prisma + Drizzle (`packages/database`) on PostgreSQL
- **Auth**: Better Auth (`packages/auth`)
- **Payments**: Stripe, LemonSqueezy, Polar, Creem, DodoPayments (`packages/payments`)
- **UI**: Tailwind CSS + shadcn/Radix (`packages/ui`)
- **Infra extras**: storage, mail, i18n, logs, notifications, AI integrations

## Key Paths

- SaaS app: `apps/saas/`
- Marketing app: `apps/marketing/`
- Shared API: `packages/api/`
- Shared DB: `packages/database/`
- Shared auth: `packages/auth/`
- Shared payments: `packages/payments/`
