AACsearch ist ein gehostetes **Search-as-a-Service**, das für E-Commerce- und CRM-Kataloge entwickelt wurde. Händler installieren ein CMS-Modul
(PrestaShop oder Bitrix), verbinden ihren AACsearch-Projekt und erhalten einen schnellen, tippfehlerresistenten Suchendpunkt sowie ein
gehostetes Widget – ohne selbst eine Typesense-Infrastruktur betreiben zu müssen.

## Projektkontext

AACsearch basiert auf einem **supastarter Next.js-Monorepo**. Die SaaS-App (`apps/saas`) stellt das Dashboard bereit,
die Marketing-Website (`apps/marketing`) dient als Landingpage, und diese Dokumentations-Website (`apps/docs`) enthält
die Dokumentation, die Sie gerade lesen.

Die Suchmaschine ist [Typesense](https://typesense.org) – eine schnelle, tippfehlerresistente Open-Source-Suchmaschine.
AACsearch ergänzt sie um Mandantenfähigkeit, Authentifizierung, Kontingentdurchsetzung, CMS-Connectoren und ein gehostetes Widget, sodass
Entwickler und Händler Typesense nicht direkt betreiben müssen.

Siehe [Architektur](/docs/overview/architecture) für das vollständige Systembild.
