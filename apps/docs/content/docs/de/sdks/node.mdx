## SearchClient (browsersicher)

### Initialisierung

```ts
import { SearchClient } from "@aacsearch/client";

const client = new SearchClient({
	baseUrl: "https://app.aacsearch.com",
	apiKey: "ss_search_...",
	indexSlug: "products",
});
```

### Einfache Suche

```ts
const result = await client.search({
	q: "running shoes",
	queryBy: "title,description",
	facetBy: "category,brand",
});

console.log(result.hits, result.found);
// { hits: [...], found: 42, page: 1, perPage: 10, facetCounts: [...] }
```

### Multi-Suche

FĂĽhren Sie mehrere Abfragen in einem einzigen Round-Trip aus:

```ts
const { results } = await client.multiSearch([
	{ q: "nike", queryBy: "title" },
	{ q: "adidas", queryBy: "title", filterBy: "in_stock:=true" },
]);
```

## AdminClient (serverseitig)

### Initialisierung

```ts
import { AdminClient } from "@aacsearch/client";

const admin = new AdminClient({
	baseUrl: "https://app.aacsearch.com",
	apiKey: "aa_admin_...",
	projectId: "org_xxx", // Ihre Organisations-ID
});
```

### Index-Verwaltung

```ts
// Index erstellen
await admin.createIndex({
	slug: "products",
	displayName: "Products",
	fields: [
		{ name: "title", type: "string" },
		{ name: "price", type: "float", facet: true },
		{ name: "category", type: "string", facet: true },
	],
	defaultSortingField: "price",
});

// Indizes auflisten
const indexes = await admin.listIndexes();

// Statistiken abrufen
const stats = await admin.getIndexStats(indexes[0].id);
```

### Dokumenten-CRUD

```ts
// Einzelnes Dokument einfĂĽgen/aktualisieren
await admin.upsertDocument("index_id", "doc_1", {
	title: "Running Shoes",
	price: 89.99,
	category: "Footwear",
});

// Batch einfĂĽgen/aktualisieren (bis zu 5000)
await admin.batchUpsertDocuments("index_id", [
	{ id: "doc_2", title: "T-Shirt", price: 29.99 },
	{ id: "doc_3", title: "Cap", price: 14.99 },
]);

// Batch lĂ¶schen
await admin.batchDeleteDocuments("index_id", ["doc_2", "doc_3"]);
```

### Synonyme & Kuratierungen

```ts
// Synonyme

	root: "shoe",
```

## Fehlerbehandlung

```ts
import { SdkError, SearchClient } from "@aacsearch/client";

const client = new SearchClient({ baseUrl, apiKey, indexSlug: "test" });

try {
	await client.search({ q: "test" });
} catch (err) {
	if (err instanceof SdkError) {
		switch (err.code) {
			case "rate_limited":
				// ZurĂĽcksetzen und erneut versuchen
				break;
			case "quota_exceeded":
				// Abrechnung kontaktieren
				break;
			case "unauthorized":
				// API-SchlĂĽssel ist ungĂĽltig
				break;
			default:
				console.error(err.message, err.details);
		}
	}
}
```

### Fehlercodes

| Code                   | Wann                                   | HTTP-Status |
| ---------------------- | -------------------------------------- | ----------- |
| `missing_bearer_token` | Kein Auth-Header                       | 401         |
| `unauthorized`         | Leeres oder ungĂĽltiges Token-Format    | 401         |
| `forbidden`            | UngĂĽltiger/abgelaufener/widerrufener   | 403         |
| `not_found`            | Ressource nicht gefunden               | 404         |
| `conflict`             | Slug bereits vergeben                  | 409         |
| `rate_limited`         | Pro-SchlĂĽssel-Ratenlimit ĂĽberschritten | 429         |
| `quota_exceeded`       | Monatsplan-Limit ĂĽberschritten         | 402         |
| `invalid_input`        | Validierungsfehler im Anfragetext      | 400         |
| `search_failed`        | Upstream-Typesense-Fehler              | 502         |
| `network_error`        | Fetch fehlgeschlagen oder Verbindung   | 0           |
