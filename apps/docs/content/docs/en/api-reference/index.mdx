# API Reference

Welcome to the AACsearch v1 REST API reference. All endpoints require Bearer token authentication using an API key.

## Authentication

All API requests must include an `Authorization: Bearer <api-key>` header. API keys are created from the dashboard or via the [keys API](/api-reference/keys/keyid).

## Endpoint Groups

- **Projects** — Manage your organizations/projects
- **Indexes** — Create and manage search indexes
- **Documents** — Index, search, and manage documents
- **Search** — Execute search queries and get suggestions
- **Analytics** — Retrieve search analytics and usage data
- **Keys** — Manage API keys
- **Synonyms** — Configure search synonyms
- **Curations** — Configure search result overrides
- **Sorting** — Manage sortable fields
- **Facets** — Manage faceted fields

Try the [Getting Started](/getting-started/overview) guide for a walkthrough, or browse the endpoint groups in the sidebar.
