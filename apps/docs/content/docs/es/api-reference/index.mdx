# Referencia de API

Bienvenido a la referencia de la API REST AACsearch v1. Todos los endpoints requieren autenticación mediante token Bearer con una clave de API.

## Autenticación

Todas las solicitudes de API deben incluir un encabezado `Authorization: Bearer ***`. Las claves de API se crean desde el panel de control o a través de la [API de claves](/api-reference/keys/keyid).

## Grupos de endpoints

- **Proyectos** — Gestionar organizaciones/proyectos
- **Índices** — Crear y gestionar índices de búsqueda
- **Documentos** — Indexar, buscar y gestionar documentos
- **Búsqueda** — Ejecutar consultas de búsqueda y obtener sugerencias
- **Analíticas** — Recuperar analíticas de búsqueda y datos de uso
- **Claves** — Gestionar claves de API
- **Sinónimos** — Configurar sinónimos de búsqueda
- **Curaciones** — Configurar anulaciones de resultados de búsqueda
- **Ordenación** — Gestionar campos ordenables
- **Facetas** — Gestionar campos facetados

Pruebe la guía [Primeros pasos](/getting-started/overview) para un tutorial, o explore los grupos de endpoints en la barra lateral.
