AACsearch es un servicio de **búsqueda como servicio** alojado, diseñado para catálogos de comercio electrónico y CRM. Los comerciantes instalan un módulo CMS
(PrestaShop o Bitrix), conectan su proyecto AACsearch y obtienen un endpoint de búsqueda rápido y tolerante a errores tipográficos, más un
widget alojado — sin necesidad de gestionar infraestructura de Typesense por su cuenta.

## Contexto del proyecto

AACsearch está construido sobre un **monorepo Next.js de supastarter**. La aplicación SaaS (`apps/saas`) proporciona el panel de control,
el sitio de marketing (`apps/marketing`) sirve la página de inicio, y este sitio de documentación (`apps/docs`) proporciona
la documentación que está leyendo ahora.

El motor de búsqueda es [Typesense](https://typesense.org) — un motor de búsqueda de código abierto rápido y tolerante a errores tipográficos.
AACsearch lo envuelve con multi-tenencia, autenticación, aplicación de cuotas, conectores CMS y un widget alojado para que
los desarrolladores y comerciantes no tengan que operar Typesense directamente.

Consulte [Arquitectura](/docs/overview/architecture) para obtener la imagen completa del sistema.
