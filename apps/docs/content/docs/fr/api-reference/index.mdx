# Référence API

Bienvenue dans la référence de l'API REST AACsearch v1. Tous les endpoints nécessitent une authentification par token Bearer utilisant une clé API.

## Authentification

Toutes les requêtes API doivent inclure un en-tête `Authorization: Bearer ***`. Les clés API sont créées depuis le tableau de bord ou via l'[API des clés](/api-reference/keys/keyid).

## Groupes d'endpoints

- **Projets** — Gérer les organisations/projets
- **Index** — Créer et gérer les index de recherche
- **Documents** — Indexer, rechercher et gérer les documents
- **Recherche** — Exécuter des requêtes de recherche et obtenir des suggestions
- **Analytiques** — Récupérer les analytiques de recherche et les données d'utilisation
- **Clés** — Gérer les clés API
- **Synonymes** — Configurer les synonymes de recherche
- **Curations** — Configurer les surcharges de résultats de recherche
- **Tri** — Gérer les champs triables
- **Facettes** — Gérer les champs à facettes

Essayez le guide [Premiers pas](/getting-started/overview) pour une introduction, ou parcourez les groupes d'endpoints dans la barre latérale.
