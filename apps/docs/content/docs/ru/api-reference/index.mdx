# Справочник API

Добро пожаловать в справочник REST API AACsearch v1. Все конечные точки требуют аутентификации с помощью Bearer-токена и ключа API.

## Аутентификация

Все API-запросы должны содержать заголовок `Authorization: Bearer ***`. Ключи API создаются в панели управления или через [API ключей](/api-reference/keys/keyid).

## Группы конечных точек

- **Проекты** — Управление организациями/проектами
- **Индексы** — Создание и управление поисковыми индексами
- **Документы** — Индексация, поиск и управление документами
- **Поиск** — Выполнение поисковых запросов и получение подсказок
- **Аналитика** — Получение аналитики поиска и данных об использовании
- **Ключи** — Управление ключами API
- **Синонимы** — Настройка синонимов поиска
- **Курации** — Настройка переопределений результатов поиска
- **Сортировка** — Управление сортируемыми полями
- **Фасеты** — Управление фасетными полями

Попробуйте руководство [Начало работы](/getting-started/overview) для ознакомления или просмотрите группы конечных точек на боковой панели.
