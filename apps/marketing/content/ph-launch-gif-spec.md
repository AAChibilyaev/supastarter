# Product Hunt Launch GIF Spec — AACsearch Widget Demo

## Format: MP4 / GIF (20s max, loopable)

**Tool:** Screen recording (QuickTime / OBS / Kap) → crop + compress

## Storyboard (20 seconds)

### 0-3s: Opening frame

- Browser window at aacsearch.com dashboard
- Cursor clicks "New Collection" button
- Overlay text: "Add search to your site in 60 seconds"

### 3-8s: Create collection + upload

- Type collection name "products"
- Drag-and-drop JSON file into upload area
- Spinner → "Indexing complete" toast
- Overlay text: "Import your data — automatic schema detection"

### 8-13s: Widget embed

- Navigate to Widget tab
- Copy embed code (highlight with cursor)
- Overlay text: "Copy one snippet → paste anywhere"

### 13-18s: Live demo

- Switch to browser tab showing demo store
- Type "wireless" in search bar
- Results appear instantly with faceted filters
- Click a product → smooth overlay
- Overlay text: "Sub-10ms search, typo-tolerant, faceted filters"

### 18-20s: Logo + CTA

- Fade to AACsearch logo
- CTA text: "Start free at aacsearch.com"
