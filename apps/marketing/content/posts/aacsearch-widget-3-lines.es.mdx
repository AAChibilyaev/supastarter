La búsqueda es la funcionalidad de mayor impacto que puedes añadir a un sitio web — y con el Widget de AACsearch, también es la más fácil. Tres líneas de HTML convierten cualquier sitio en una experiencia de búsqueda con autocompletado, navegación por teclado y filtrado facetado.

## La Configuración de 3 Líneas

Añade estas tres líneas a cualquier página HTML:

```html
<script src="https://cdn.aacsearch.com/widget.js"></script>
<link rel="stylesheet" href="https://cdn.aacsearch.com/widget.css" />
<div id="search-root"></div>
```

Luego, en una etiqueta script o en tu archivo principal de JavaScript:

```javascript
new AACsearchWidget({
	apiKey: "ss_search_your_api_key",
	index: "products",
	mountElement: "#search-root",
});
```

Eso es todo. Tu sitio ahora tiene una barra de búsqueda completa que devuelve resultados mientras el usuario escribe.

## Cómo Funciona

El Widget se conecta directamente a la API de AACsearch desde el navegador. Cuando un usuario escribe, el Widget:

1. Envía la consulta a AACsearch con tu API key e índice
2. Recibe los resultados coincidentes (típicamente en menos de 50ms)
3. Renderiza una superposición con tarjetas de producto, fragmentos o plantillas personalizadas
4. Maneja la navegación por teclado (flechas, Enter, Escape)
5. Registra eventos de clic para analíticas

Sin servidor backend, sin paso de compilación, sin sincronización de base de datos. El Widget maneja cada interacción.

## Lo Que Obtienes de Serie

El Widget incluye estas funcionalidades activas:

| Característica           | Descripción                                                                         |
| ------------------------ | ----------------------------------------------------------------------------------- |
| Autocompletado           | Los resultados aparecen mientras escribes, con debounce de 300ms                    |
| Navegación por teclado   | Flechas para navegar, Enter para seleccionar, Escape para cerrar                    |
| Tolerancia a errores     | Consultas mal escritas aún devuelven resultados relevantes                          |
| Fragmentos de resultados | Muestra título, extracto, miniatura y precio (configurable)                         |
| Modo oscuro              | Detecta automáticamente la preferencia del sistema, variables CSS para personalizar |
| Adaptable a móviles      | Ancho completo en móvil, cajón inferior para facetas en pantallas pequeñas          |
| Analíticas               | Registra consultas de búsqueda, clics y búsquedas sin resultados                    |
| Filtros facetados        | Muestra paneles de facetas cuando los campos están configurados                     |

## Ejemplos de Integración

### Sitio HTML Estático

```html
<!DOCTYPE html>
<html>
	<head>
		<script src="https://cdn.aacsearch.com/widget.js" defer></script>
		<link rel="stylesheet" href="https://cdn.aacsearch.com/widget.css" />
	</head>
	<body>
		<header>
			<div id="search-root"></div>
		</header>
		<script>
			document.addEventListener("DOMContentLoaded", () => {
				new AACsearchWidget({
					apiKey: "ss_search_demo_key",
					index: "docs",
					placeholder: "Search documentation...",
					mountElement: "#search-root",
				});
			});
		</script>
	</body>
</html>
```

### WordPress (functions.php)

```php
function add_aacsearch_widget() {
    wp_enqueue_script('aacsearch-widget', 'https://cdn.aacsearch.com/widget.js', [], null, true);
    wp_enqueue_style('aacsearch-widget', 'https://cdn.aacsearch.com/widget.css');
}
add_action('wp_enqueue_scripts', 'add_aacsearch_widget');

function render_aacsearch_widget() {
    echo '<div id="search-root"></div>';
    echo '<script>
        document.addEventListener("DOMContentLoaded", function() {
            new AACsearchWidget({
                apiKey: "' . esc_js(get_option('aacsearch_api_key')) . '",
                index: "wordpress_posts",
                mountElement: "#search-root"
            });
        });
    </script>';
}
add_action('wp_footer', 'render_aacsearch_widget');
```

### React / Next.js

```tsx
"use client";

import { useEffect, useRef } from "react";

export function AACSearchWidget() {
	const ref = useRef<HTMLDivElement>(null);

	useEffect(() => {
		if (!ref.current) return;

		const widget = new AACsearchWidget({
			apiKey: process.env.NEXT_PUBLIC_AACSEARCH_API_KEY!,
			index: "blog",
		});

		widget.mount(ref.current);
		return () => widget.destroy();
	}, []);

	return <div ref={ref} />;
}
```

### Vue.js

```vue
<template>
	<div ref="searchRoot"></div>
</template>

<script setup>
import { ref, onMounted, onUnmounted } from "vue";

const searchRoot = ref(null);
let widget = null;

onMounted(() => {
	widget = new AACsearchWidget({
		apiKey: "ss_search_demo_key",
		index: "products",
		mountElement: searchRoot.value,
	});
});

onUnmounted(() => widget?.destroy());
</script>
```

## Configuración Avanzada

El Widget acepta estas opciones para un control detallado:

```javascript
const search = new AACsearchWidget({
	// Required
	apiKey: "ss_search_...",
	index: "products",

	// Display
	placeholder: "Search products...",
	theme: "light", // 'light' | 'dark' | 'auto'
	resultsPerPage: 12,
	maxSuggestions: 5,

	// Search behavior
	searchableFields: ["name", "brand", "description", "sku"],
	facetFields: ["brand", "category", "price"],
	sortOptions: [
		{ label: "Relevance", value: "_text_match:desc" },
		{ label: "Price: Low to High", value: "price:asc" },
	],

	// Custom rendering
	resultTemplate: (item) => `
    <div class="aacsearch-result">
      <img src="${item.image}" alt="${item.name}" />
      <div class="aacsearch-result-info">
        <h3>${item.name}</h3>
        <p class="price">$${item.price}</p>
        <span class="brand">${item.brand}</span>
      </div>
    </div>
  `,

	// Analytics
	trackClicks: true,
	onSearch: (query, results) => console.log("Searched:", query, "Results:", results.length),
	onClick: (result) => console.log("Clicked:", result.name),
});
```

## Rendimiento

- **Primera renderización:** < 200ms (el JS del Widget es ~35KB comprimido con gzip)
- **Respuesta de consulta:** < 50ms (P95 desde US West)
- **Renderización de resultados:** < 20ms (scroll virtual para conjuntos grandes de resultados)
- **Tamaño del bundle:** 35KB JS + 8KB CSS (comprimido con gzip)

## Limitaciones

El Widget está diseñado para búsqueda — no para construir una tienda completa. Úsalo para:

- Búsqueda en sitios de documentación (mejor ajuste)
- Búsqueda de productos de comercio electrónico
- Búsqueda de blogs / contenido
- Paleta de comandos para apps SaaS

Para interfaces complejas (páginas de producto personalizadas, flujos de pago), usa la API REST de AACsearch directamente.
