# Как AgencyHub предоставляет корпоративный поиск для 12 клиентов по ценам стартапа

> _«С AACsearch мы настраиваем поиск для нового клиента за 15 минут. Каждый клиент думает, что у него собственная выделенная поисковая инфраструктура — и это действительно так. Мы просто управляем всем с единой панели управления.»_
> — Маркус Вебер, Технический директор AgencyHub

---

## Метаданные

- **Компания:** AgencyHub — полносервисное цифровое агентство, обслуживающее средне-рыночные e-commerce бренды
- **Отрасль:** Агентство / Веб-разработка
- **Масштаб:** 12 активных клиентов, 500 000+ проиндексированных товаров, 800 000+ поисковых запросов в месяц
- **Предыдущее решение:** Смешанное — некоторые клиенты на Algolia, некоторые на самостоятельно размещённом Elasticsearch, некоторые на встроенном поиске Shopify
- **Вариант использования:** Поиск под управлением агентства для клиентов e-commerce
- **AACsearch с:** Ноябрь 2025

---

## 1. Проблема

AgencyHub управляет сайтами электронной коммерции для 12 средне-рыночных брендов, каждый с 5 000–100 000 товаров. Их поисковая инфраструктура представляла собой фрагментированный беспорядок:

**Клиенты Algolia (5).** У каждого клиента была своя учётная запись Algolia с отдельным выставлением счетов. Итого: ~$2 500/месяц. Агентству приходилось входить в 5 разных панелей управления Algolia для мониторинга качества поиска.

**Клиенты Elasticsearch (3).** AgencyHub использовал общий кластер Elasticsearch и применял логику на уровне приложения для разделения данных клиентов. Любое изменение схемы рисковало повлиять на всех клиентов в кластере.

**Встроенный поиск Shopify (4).** Эти клиенты использовали встроенный поиск Shopify — ограниченные возможности, никакой аналитики, и агентство не могло его оптимизировать.

«Накладные расходы убивали нас, — говорит Маркус Вебер, Технический директор AgencyHub. — Мы управляли 5 учётными записями Algolia, хрупким кластером Elasticsearch и не могли продать улучшения поиска нашим клиентам Shopify, потому что нам нечего было предложить.»

---

## 2. Решение

AgencyHub объединил всех клиентов на AACsearch по трём причинам:

1. **Встроенная мультитенантность** — каждый клиент получает изолированные поисковые коллекции с ограниченными API-токенами, всё в одной учётной записи
2. **Готовность для white-label** — встраиваемый виджет полностью настраивается под бренд каждого клиента
3. **Аналитика по каждому клиенту** — отдельная аналитика для каждого клиента, показывающая объём поиска, основные запросы и коэффициент нулевых результатов

Миграция выполнялась последовательно по клиентам в течение двух недель:

- **Миграция Algolia:** Использование импортного API AACsearch для прямой миграции. В среднем: 2 часа на клиента.
- **Миграция Elasticsearch:** Экспорт данных каждого клиента и импорт в отдельные коллекции. В среднем: 3 часа на клиента.
- **Обновление Shopify:** Для 4 клиентов Shopify AgencyHub установил виджет AACsearch в качестве поискового оверлея, заменив встроенный поиск Shopify. В среднем: 1 час на клиента.
- **White-label брендинг:** Настройка CSS, цветов и шрифтов поискового виджета для фирменного стиля каждого клиента.

«Каждый клиент получил подробный документ о миграции. Никто из них не заметил простоев», — говорит Вебер.

## 3. Результаты

| Показатель                              | До (Смешанный)     | После (AACsearch) | Улучшение            |
| --------------------------------------- | ------------------ | ----------------- | -------------------- |
| Ежемесячная стоимость поиска (все)      | ~$3 800            | $499 (Тариф Pro)  | **Снижение в 7,6×**  |
| Время подключения нового клиента        | 3–5 дней           | 15 минут          | **На 99 % быстрее**  |
| Задержка поиска (средняя по клиентам)   | 80–200 мс          | 8–15 мс           | **В 10× быстрее**    |
| Панели клиентов для мониторинга         | 9 отдельных входов | 1 панель          | **Единая панель**    |
| Удовлетворённость поиском (от клиентов) | 3,8/5 (среднее)    | 4,6/5             | **+21 %**            |
| Тикеты поддержки по поиску              | 12/месяц           | 2/месяц           | **Снижение на 83 %** |

**Суть:** AgencyHub сократил расходы на поисковую инфраструктуру с $3 800/месяц до $499/месяц, одновременно улучшив производительность поиска и добавив аналитику для всех 12 клиентов — включая 4, у которых ранее был только базовый поиск Shopify.

## 4. Модель агентства

AgencyHub теперь предлагает «Корпоративный поиск» как пакетную дополнительную услугу:

- **Плата за настройку:** $500 за клиента (единоразово)
- **Ежемесячная плата за управление:** $200 за клиента
- **Стоимость для клиента:** Ниже, чем каждый клиент платил бы индивидуально за Algolia или Elasticsearch

Это приносит $2 400/месяц регулярного дохода для AgencyHub, в дополнение к $499, которые они платят за тариф Pro — маржа в 5×.

«Мы превратили поиск из центра затрат в центр прибыли, — говорит Вебер. — Наши клиенты получают лучший поиск, чем могли бы создать сами, и мы зарабатываем на этом.»

## 5. Технические детали

- **Проиндексированные документы:** 500 000+ товаров в 12 коллекциях
- **Используемые функции:** Мультитенантность (ограниченные API-токены), фасетный поиск, устойчивость к опечаткам, пользовательская сортировка, аналитика, white-label виджет
- **Интеграции:** Пользовательский API для обновлений товаров в реальном времени, приложение Shopify, HTML/JS виджет
- **Специфическая конфигурация агентства:** Каждая клиентская коллекция имеет пользовательские правила сортировки на основе их категорий товаров и приоритетных полей
- **График миграции:** 14 дней в целом (все 12 клиентов мигрированы без простоев)
